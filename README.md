Bu proje, bir araç kiralama şirketinin temel ihtiyaçlarını karşılamak amacıyla geliştirilmiştir. Admin ve Kullanıcı (User) olmak üzere iki farklı panelden oluşur.
Veriler yerel bir JSON dosyasında tutulur, bu sayede kurulum gerektirmeden çalışır. Kod yapısı modüler (parçalı) olarak tasarlanmış, arayüz, veri tabanı ve ayarlar birbirinden ayrılmıştır.
